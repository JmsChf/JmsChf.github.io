Snake Game History


Snake Game v1.0

The original version of the snake game was developed in 1976 by Gremlin Industries. It was a simple game where the player controlled a snake that moved around the screen, eating apples and growing longer with each apple consumed. The game ended if the snake collided with itself or the boundaries of the screen.

Snake Game v2.0

In the 1990s, the snake game gained popularity with the advent of mobile phones. It became one of the most popular games on early Nokia phones, commonly known as "Snake" or "Snake II". The gameplay remained similar to the original version, but the graphics improved and additional features were added, such as different levels of difficulty.

Snake Game v3.0

With the advancement of technology, snake games started to appear on various platforms, including web browsers. These web-based snake games often added new elements, such as power-ups, different game modes, and multiplayer capabilities. They also featured more sophisticated graphics and sound effects.

My Snake Game

My snake game demo is the most like version 1, except the snake can collide with itself and the game will continue. The game only ends when the snake collides with a wall.
